Entertainment Web App

In this project, I combined my creativity and technical skills. I have created an appealing multi-page entertaining web application. Guided by JSON data, dynamic routing, state management, and search functionality, I brought this innovative concept to life, creating a complete full-stack web application that seamlessly integrates various technologies.

View Code View the site

Key Achievements

  • Through routing, I've enabled seamless navigation between Home, Movies, TV Series, and Bookmarked pages.
  • State management allow users to add and remove Movies and TV Series from bookmarks.
  • I have implemented a search functionality on all pages, allowing users to easily discover relevant Movies and TV Series.
  • Using NextJs, TypeScript, SCSS, SCSS, Prisma, MongoDB, and NextAuth for authentication, I've created this project as a full-stack web application. This demonstrates my expertise in building end-to-end solutions.
  • I Integrated authentication pages with registration and login features, for application security and user experience.
  • Tested on various devices, browsers, and screen sizes to ensure a seamless and appealing user experience.

Tools Used

  • NextJs
  • TypeScript
  • SCSS
  • GraphQL
  • Prisma
